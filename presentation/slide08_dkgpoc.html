DKGPoC
Neuroscience knowledge assistant powered by graph RAG
R&D prototype · retrieval-first QA

Essence: answer biomedical questions with explainable, source-cited evidence from a Neo4j knowledge graph

I built DKGPoC to validate a robust GraphRAG pattern for neuroscience and psychopharmacology research. Instead of generic text search, the assistant first classifies question intent, then retrieves high-signal claims and entities from the graph, and finally generates citation-grounded answers.

Research question Query analysis Neo4j evidence retrieval Cited answer
  • Retriever design — custom LangChain retriever that maps each question to claim/entity/mechanism search, uses Neo4j full-text index when available, and falls back to resilient Cypher matching when needed.
  • Evidence traceability — every answer is linked to structured metadata (claim text, confidence, DOI/document, section/page), making outputs auditable for research workflows.
