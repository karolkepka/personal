Wideoo (Memeez)
AI content studio: animate photo, music, graphics and video workflows
Product prototype · full-stack ownership

Essence: one platform to generate multimodal content with guardrails, async processing and user-ready outputs

I built Wideoo as an end-to-end personal product to prove delivery across product, backend, frontend and operations. The app combines AI providers to generate music, graphics/memes, and image-to-video animations, then exposes results in a simple user workflow with downloadable assets and history.

Prompt + media input AI generation Async task status Gallery + download
  • Architecture — React frontend + FastAPI backend, SQLAlchemy persistence for users/photos, local object storage pattern for generated files, and Celery-based scheduled housekeeping.
  • Operational discipline — model configuration by domain (music/graphics/animation), policy guardrails, structured observability (Grafana/Loki dashboards), and containerized runtime with Docker Compose.
